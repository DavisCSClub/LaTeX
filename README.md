# latex

Compiling on Linux
--------------------------
1. Make sure that both pdflatex and bibtex are installed.
2. Follow the instructions at http://tex.stackexchange.com/a/37731 :
	run pdflatex, bibtex, pdflatex, pdflatex, in that order.
